Les décorateurs

En plus de Python de base, vous devez également être conscient de ce concept plus avancé en Python.

Les fonctions, et à peu près tout le reste en Python, sont des objets comme int or string.

Parce qu'ils sont des objets, vous pouvez faire plusieurs choses avec eux, à savoir :

Vous pouvez passer une fonction comme argument à une autre fonction de la même manière que vous passez un string or int comme argument de la fonction.

Les fonctions peuvent également être renvoyées par d'autres fonctions comme vous renverriez d'autres string or int valeurs.

Les fonctions peuvent être stockées dans des variables.

En fait, la seule différence entre les objets fonctionnels et les autres objets est que les objets fonctionnels contiennent la méthode magique __call__().

Un décorateur Python est simplement une fonction qui prend une fonction en argument et renvoie une version modifiée de la fonction qui a été transmise.
