Améliorer une application Web Python par des tests et du débogage



Dans ce projet, votre mission consiste à améliorer une application Web développée avec le microframework Flask. Vous devez déboguer le code et gérer les erreurs et exceptions en Python. Vous allez découvrir et adopter la méthode TDD (Test-Driven Development), où le développement se fait de manière itérative en alternant entre l’écriture des tests et du code. Vous développerez également une suite de tests Python avec pytest et des tests automatisés avec Selenium pour vous assurer que le code est intégralement couvert par les tests et qu'il est de bonne qualité.
